# Commands

## Comandos principais

- `agentforge install`
- `agentforge status`
- `agentforge add-agent`
- `agentforge add-flow`
- `agentforge add-engine`
- `agentforge validate`
- `agentforge export`
- `agentforge update`
- `agentforge uninstall`

## Quando usar

- Instalar a base: `install`
- Verificar o estado: `status`
- Criar equipe e fluxos: `add-agent`, `add-flow`
- Validar a estrutura: `validate`
- Gerar derivados: `export`
