# Agent Design Playbook

## Objetivo

Definir a equipe AgentForge adequada ao projeto.

## O que fazer

1. Ler o contexto mínimo do projeto.
2. Escolher agentes e subagentes relevantes.
3. Justificar cada escolha com sinais do repositório.
4. Registrar o checkpoint ao concluir.

## Saída esperada

- Lista de agentes sugeridos.
- Justificativa curta por agente.
- Critérios de conclusão.
