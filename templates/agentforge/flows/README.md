# Flows

Os fluxos descrevem como a equipe AgentForge trabalha.
Cada arquivo desta pasta explica um caminho operacional legível para humanos e fácil de transformar em rotina.

## O que colocar aqui

- Objetivo do fluxo.
- Quando usar.
- Etapas.
- Checkpoints.
- Aprovações humanas.
- Condição de término.
