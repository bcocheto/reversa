# Glossary

## Termos

| Termo | Significado |
| --- | --- |
| Agent | Papel operacional que atua em uma etapa do trabalho. |
| Subagent | Especialista acionado para um assunto mais estreito. |
| Flow | Sequência operacional com etapas e checkpoints. |
| Policy | Regra de segurança, proteção ou aprovação. |
| Harness | Camada que orienta leitura e roteamento. |
| Memory | Decisões, lições e perguntas abertas do projeto. |

## Observações

- Atualize a lista quando surgir uma palavra recorrente nova.
