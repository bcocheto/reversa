# Policies

As políticas controlam o que a equipe AgentForge pode ler, escrever e quando precisa pedir aprovação humana.

## Use para

- Proteger arquivos críticos.
- Definir limites de escrita.
- Registrar aprovações obrigatórias.

## Regra básica

- Se houver dúvida de segurança, pare e peça confirmação.
