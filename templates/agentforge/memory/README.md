# Memory

Esta pasta guarda o histórico que vale a pena manter entre sessões.
Use-a para decisões, aprendizados e perguntas que ainda não foram resolvidas.

## O que entra aqui

- Decisões já tomadas.
- Lições aprendidas.
- Questões abertas.

## Como manter

- Escreva de forma curta.
- Inclua contexto suficiente para alguém retomar depois.
