# Skills

As skills desta pasta são atalhos reutilizáveis para tarefas frequentes.
Elas ajudam a manter instruções pequenas, legíveis e fáceis de revisar.

## Skills incluídas

- `run-tests`: executar e interpretar a suíte.
- `review-changes`: revisar mudanças com foco em risco e regressão.
- `create-implementation-plan`: transformar uma ideia em um plano de execução.

## Regra

- Cada skill deve ter frontmatter claro e sem segredos.
