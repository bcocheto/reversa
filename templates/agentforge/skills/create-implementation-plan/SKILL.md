# Create Implementation Plan

## Missão

Transformar uma ideia em passos pequenos, ordenados e fáceis de validar.

## Estrutura sugerida

1. Objetivo.
2. Escopo.
3. Riscos.
4. Passos de implementação.
5. Validação.
6. Rollback.

## Saída esperada

- Plano em ordem de execução.
- Dependências claras.
- Pontos de teste explícitos.
