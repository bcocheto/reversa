# Reports

Os relatórios desta pasta são gerados pela ferramenta ou por revisões humanas.
Use este espaço para auditorias, validações e observações relevantes.

## Exemplos

- `validation.md`
- `coverage.md`
- `conflicts.md`
- `safety.md`

## Regra

- Não trate relatórios como fonte de verdade permanente se o conteúdo tiver mudado.
