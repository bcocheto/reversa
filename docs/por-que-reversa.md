# Historical context: Reversa

This page is kept as legacy context.

Reversa was the predecessor name for the original reverse-engineering workflow that eventually evolved into AgentForge. The current product no longer centers on a spec dump alone. It creates and maintains the agent-ready layer of a project.

If you are looking for the current product story, start at [Why AgentForge exists](por-que-agentforge.md).

## What changed

- The source of truth moved to `.agentforge/`.
- Bootloaders became managed engine exports.
- Context is routed through a harness and a context index.
- Skills, flows, policies, memory, and reports are first-class canonical files.

## Why keep this page

- to preserve the historical name for existing users
- to avoid hiding where the project came from
- to make it clear that AgentForge is the current product, not just a rename
