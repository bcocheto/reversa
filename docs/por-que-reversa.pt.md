# Contexto histórico: Reversa

Esta página fica como contexto legado.

Reversa foi o nome anterior do fluxo original de reverse-engineering que acabou evoluindo para o AgentForge. O produto atual não gira mais em torno de um dump de specs. Ele cria e mantém a camada agent-ready de um projeto.

Se você quer a história do produto atual, comece em [Por que o AgentForge existe](por-que-agentforge.md).

## O que mudou

- A fonte da verdade passou para `.agentforge/`.
- Bootloaders viraram exports gerenciados por engine.
- O contexto passou a ser roteado por harness e context index.
- Skills, flows, policies, memory e reports viraram arquivos canônicos de primeira classe.

## Por que manter esta página

- preservar o nome histórico para usuários antigos
- deixar claro de onde o projeto veio
- mostrar que AgentForge é o produto atual, não só um rename
